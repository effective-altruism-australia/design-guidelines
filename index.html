Effective Altruism Australia Design Guidelines

Color Palette

#0C869B

Primary Teal

Usage: Accents, buttons, links, highlights

✓ Use for interactive elements and brand accents

✗ Don't use as a background for large areas

#025159

Dark Teal

Usage: Headings, dark text, navigation

✓ Use for headings and important text elements

✗ Don't use on dark backgrounds without sufficient contrast

#F4EFE9

Warm Off-White

Usage: Page backgrounds, subtle sections

✓ Use as primary background color

✗ Don't use for text or small elements

#151515

Charcoal Black

Usage: Body text, primary content

✓ Use for main body text and content

✗ Don't use as background without white text

#FFFFFF

Pure White

Usage: Card backgrounds, content areas

✓ Use for content cards and clean backgrounds

✗ Don't use for text on light backgrounds

Utilities

Typography

Merriweather

Usage: Headings (H1, H2, H3, etc.)

✓ Use for all headings and titles

✗ Don't use for body text or long paragraphs

A classic serif font that provides elegance and readability for headings.

Mukta

Usage: Body text, paragraphs, UI elements

✓ Use for all body text and interface elements

✗ Don't use for headings or titles

A clean, modern sans-serif font optimized for readability in body text.

Default Sans-Serif

Usage: Email and limited contexts

✓ Use when custom fonts aren't available (emails, etc.)

✗ Don't use when Merriweather and Mukta are available

Fallback to system default sans-serif fonts when custom fonts cannot be loaded.

Bad Examples - What NOT to Do

Mistake #1

✗ Stack the text next to the logo

✓ Use standard logo with two lines of text

Mistake #2

✗ Centre-justify the text next to the logo

✓ Use standard logo with right-aligned text

Mistake #3

✗ Use other fonts with the logo

✓ Use Merriweather font
